# Claude Code 斜杠命令速查表

## mattpocock/skills 工程技能

| 斜杠命令 | 功能 |
|----------|------|
| `/diagnose` | 严格诊断循环：复现 → 最小化 → 假设 → 检测 → 修复 → 回归测试 |
| `/grill-with-docs` | 深入访谈挑战计划，更新 `CONTEXT.md` 和 ADR |
| `/improve-codebase-architecture` | 发现代码库深化机会，改善架构 |
| `/prototype` | 构建可丢弃原型以充实设计 |
| `/setup-matt-pocock-skills` | 初始化 per-repo 配置（issue tracker、标签词汇、文档布局） |
| `/tdd` | 红-绿-重构测试驱动开发 |
| `/to-issues` | 将计划/SPEC/PRD 拆分为独立的 GitHub issues（垂直切片） |
| `/to-prd` | 将当前对话上下文转为 PRD 并提交为 GitHub issue |
| `/triage` | 通过状态机对 issue 进行分类 |
| `/zoom-out` | 放大视角，在更广上下文中解释代码 |

## mattpocock/skills 生产力技能

| 斜杠命令 | 功能 |
|----------|------|
| `/caveman` | 超压缩沟通模式，减少约 75% token 使用 |
| `/grill-me` | 深入访谈，厘清计划或设计的每个分支 |
| `/handoff` | 将当前对话压缩为 handoff 文档，以便其他 agent 继续 |
| `/write-a-skill` | 创建具有正确结构、渐进披露和捆绑资源的新技能 |

## mattpocock/skills 杂项技能

| 斜杠命令 | 功能 |
|----------|------|
| `/git-guardrails-claude-code` | 设置 Claude Code hooks 阻止危险 git 命令 |
| `/migrate-to-shoehorn` | 将测试文件从 `as` 类型断言迁移到 @total-typescript/shoehorn |
| `/scaffold-exercises` | 创建练习目录结构（sections、problems、solutions、explainers） |
| `/setup-pre-commit` | 设置 Husky pre-commit hooks（lint-staged、Prettier、类型检查、测试） |

## Claude Code 内置命令

| 斜杠命令 | 功能 |
|----------|------|
| `/help` | 获取 Claude Code 使用帮助 |
| `/clear` | 清除当前会话 |
| `/config` | 配置 Claude Code 设置 |

## Claude Code 内置技能（通过 Skill 工具调用）

| 技能名称 | 功能 |
|----------|------|
| `claudeception` | 从工作会话中提取可复用的知识到 Claude Code 技能 |
| `claude-api` | 构建、调试、优化 Claude API 应用（含 prompt 缓存） |
| `verify` | 验证代码变更是否按预期工作 |
| `simplify` | 审查更改的代码以实现重用、质量和效率 |
| `harness` | 多会话自主代理工作，需要进度检查点、失败恢复、任务依赖管理 |
| `loop` | 按固定间隔运行提示或斜杠命令 |
| `run` | 启动并驱动项目查看运行效果 |
| `init` | 初始化新的 CLAUDE.md 文件 |
| `review` | 审查 Pull Request |
| `security-review` | 对当前分支的待处理更改进行安全审查 |
| `skills-vetter` | 技能安全审核工具，验证恶意代码、敏感信息等 |
| `save-summary` | 保存当前会话摘要到记忆库 |
| `systematic-debugging-skill` | 四阶段调试框架：根因调查 → 才能尝试修复 |
| `update-config` | 配置 harness 行为、权限、环境变量等 |
| `keybindings-help` | 自定义键盘快捷键、重绑键、添加和弦绑定 |
| `xiaohongshu` | 小红书内容发布和管理技能 |
| `browser-tool-automation` | 通过 Rube MCP 自动化浏览器任务 |
| `fewer-permission-prompts` | 扫描 transcripts 中的常见只读调用，添加到 allowlist |
